# 🛒 Ecommerce Sales Analysis (Python)

An end-to-end ecommerce data analysis project using **Python**, focused on data cleaning, feature engineering, and extracting meaningful business insights.

---

## 📌 Project Overview
This project analyzes ecommerce sales data to uncover trends, customer behavior, and product performance.  
It demonstrates real-world data analysis workflow from raw data to insights.

---

## 🚀 Key Features
- Data Cleaning & Preprocessing  
- Feature Engineering  
- Sales Trend Analysis  
- Top Products Identification  
- Customer Purchase Behavior  
- Data Visualization using charts  

---

## 🛠️ Technologies Used
- Python  
- pandas  
- NumPy  
- Matplotlib
