# 🛒 Ecommerce Sales Analysis (Python)

An end-to-end ecommerce data analysis project using **Python**, designed to extract actionable business insights from raw sales data.  
This project showcases **real-world data analysis workflow**, including data cleaning, feature engineering, and advanced analytics.

---

## 📌 Project Overview
This project analyzes ecommerce sales data to uncover trends, customer behavior, and product performance.  
It demonstrates a professional data analysis workflow:

1. Data Cleaning & Preprocessing  
2. Feature Engineering (Revenue, Discounts, Net Revenue, Customer ID)  
3. Customer Lifetime Value (CLV) and Top 20% Customers (Pareto Analysis)  
4. Email Domain Analysis  
5. Product & City Performance  
6. Data Visualization (Monthly Sales, Top Products)  
7. Exporting Cleaned Data for Reporting

The workflow is designed to mimic **industry-level data analysis pipelines** used by data analysts and business intelligence teams.

---

## 🚀 Key Features
- ✅ Data Cleaning & Preprocessing (handling missing values, outliers, duplicates)  
- ✅ Feature Engineering (Revenue, Discounts, Tax, Net Revenue)  
- ✅ Customer Analysis (CLV, Top 20% customers, Email domains)  
- ✅ Sales Trend Analysis (Monthly Sales, Product & City performance)  
- ✅ Data Visualization with clean, professional charts
